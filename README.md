# Account ERP Frontend

React, Tailwind CSS, and Framer Motion frontend for the Spring Boot accounting backend in this repository.

## Run

Install dependencies once:

```powershell
cd frontend
npm install
```

Start the dev server:

```powershell
npm run dev
```

Open:

```text
http://localhost:4173
```

The backend defaults to:

```text
http://localhost:8081
```

Set the backend URL, bearer token, company ID, and user ID from the API screen.
"# kaserafrontend"
